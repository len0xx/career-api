# Термины

Описание объектов типа термин

### Структура объекта:

Название поля | Тип данных | Описание
------------ | ------------- | -------------
id | integer | Идентификатор термина
title | string | Текстовое название термина

### Пример термина:
```javascript
{
    "id": 612,
    "title": "Екатеринбург"
}
```
